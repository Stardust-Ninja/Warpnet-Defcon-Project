Een fysieke, DEFCON-geïnspireerde dreigingsindicator voor gebruik binnen een Security Operations Center (SOC).

Dit project maakt gebruik van een Raspberry Pi Pico om het actuele dreigingsniveau visueel weer te geven via een 5-staps LED-systeem. Securitydata (zoals firewall-events, antivirusstatus, CVE-scores en actieve alarmen) wordt via een API verwerkt en vertaald naar één duidelijk DEFCON-niveau.

Ontworpen om in één oogopslag de securitystatus zichtbaar te maken binnen een SOC-omgeving.
